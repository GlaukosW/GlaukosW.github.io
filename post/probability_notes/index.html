概率论与数理统计（一）

Tue, Feb 18, 2020

第一章 概率的基本概念

随机试验

随机试验：如果一个实验具有重复性（试验可在相同条件下重复进行）、多样性（试验的结果不止一个，并且一切可能的结果都已知）、随机性（每次试验前，结果不确定），则称该实验为随机试验，简称试验。通常用 $E$ 表示。

样本空间，随机事件

样本空间, 样本点：随机试验 $E$ 所有可能的结果的全体组成的集合称为 $E$ 的样本空间，记为 $S$ 。 $S$ 中的元素（可能结果）称为样本点，$S = {e}$。注意： 样本空间的元素是由实验目的所确定的，样本空间不一定是有限的，也可以是无限的。

随机事件：称E的样本空间 $S$ 的子集为 $E$ 的随机事件，简称事件。

基本事件：由一个样本点组成的单点集（相对于观察目的不可再分解的事件）。

当且仅当集合 $A$ 中的一个样本点出现时，称事件 $A$ 发生。

必然事件：样本空间 $S$

不可能事件：空集中不包含任何样本点，每次试验必定不发生，$\phi$

事件的运算

和事件: $A\cup B = {x \in A 或x\in B}$ 称为 $A$ 与 $B$ 的和事件. 当且仅当 $A$ 与 $B$ 中至少有一个发生时, 称为 $A\cup B$ 发生. 也记为 $A +B$

类似的, 称 $\bigcup_{k = 1}^{n}A_k$ 称为 事件 $A_1,A_2 … A_n$ 的和事件, $\bigcup_{k = 1}^{\infin}A_k$ 称为可列事件 $A_1, A_2…$ 的和事件.

可列：1. 无穷多个 2.可以排列

积事件: $A\cap B = {x \in A 且x\in B}$. 也记为 $AB$

差事件: $A - B = {x\in A 且 x\notin B}$

事件间的关系

包含关系：$B\subset A$，则称 $A$ 包含 $B$ 或 $B$ 包含在 $A$中。（若 $B$ 发生则 $A$ 一定发生）。若 $B\subset A$ 且 $A\subset B$，则称 $A$ 与 $B$ 相等。

事件的互不相容性：若 $A$ 与 $B$ 不可能同时发生，则称 $A$ 与 $B$ 互不相容(互斥). $A\cap B = \phi$

对立事件：$A$ , $B$ 有且仅有一个发生, $A\cup B = S 且 AB = \phi$, 则称 $A$ 与 $B$ 互为逆事件或对立事件. $\overline{A} = S - A$

对立事件与互不相容事件:

  • 两事件对立,则一定互不相容
  • 互不相容适用于多个事件, 而对立适用于两个事件
  • 互不相容表示事件不能同时发生,但可以都不发生, 而对立事件一定有且仅有一个发生.

完备事件组: $A_1,A_2…A_n$互不相容且$A_1\cup A_2… \cup A_n = S$

事件运算律

  • 交换律: $A \cup B = B \cup A,\space A\cap B = B \cap A$.
  • 结合律: $(A \cup B) \cup C = A \cup (B \cup C), \space (A\cap B) \cap C = A \cap (B \cap C)$.
  • 分配律: $(A \cup B) \cap C =( A \cap C) \cup (B \cap C), \space (A\cap B) \cup C = (A \cup C)\cap (B \cup C)$.
  • De-Morgan律: $\overline{A\cup B}=\overline{A}\cap\overline{B},\space \overline{A\cap B}=\overline{A}\cup\overline{B}$.
    • 推广有$\overline{A_1\cup A_2…\cup A_n}=\overline{A_1}\cap\overline{A_2}…\cap\overline{A_n},\space \overline{A_1\cap A_2…\cap A_n}=\overline{A_1}\cup\overline{A_2}…\cup\overline{A_n}$.
  • $\overline{\overline{A}} = A$.
  • $A-B = A\overline{B} = A - AB$.

频率与概率

频数与频率

频数, 频率: 设在$n$次重复试验中, 事件 $A$ 出现了 $n_A$ 次,则称 $n_A$ 为事件 $A$ 在 $n$ 次试验中出现的频数, $\frac{n_A}{n}$为事件 $A$ 在 $n$ 次试验中痴线的频率, 记为 $f_n(A)$.

频率的性质:

  • 非负性: $0\leq f_n(A) \leq1$.
  • 规范性: 若 $S$ 为必然事件,则 $f_n(S) = 1$.
  • 有限可加性: 若 $A_1, A_2…A_k$ 为两两互不相容的事件,则$f_n(\bigcup_{i = 1}^kA_i) = \sum_{i=1}^kf_n(A_i)$.

频率的稳定性: 随着试验次数的增加, 事件的频率将在某个数值 $p$ 附近稳定地摆动, 此数值可以定义为事件 $A$ 发生的概率, 称为统计概率.

概率的公理化定义：

设 $E$ 为随机试验, $S$ 是其样本空间, 对 $E$ 的每个事件 $A$ 赋予一个实数, 记为 $P(A)$ , 称为事件 $A$ 的概率, 若集合函数 $P(\cdot)$满足:

  1. 非负性: $\forall A \subset S, P(A) \geq 0$
  2. 归一性: $P(S) = 1$
  3. 可列可加性:对于两两互不相容事件$A_1,A_2,…$, 有 $P\left(\bigcup_{i=1}^{\infin}A_i\right) = \sum_{i = 1}^{\infin}P(A_i)$

概率的性质:

  • $P(\phi) = 0$.
  • 有限可加性: 若$A_1, A_2…,A_n$两两互不相容, 则有$P(A_1+A_2+…+A_n) = P(A_1)+P(A_2)+…+P(A_n)$.
